# БетоНина

Одностраничный сайт-визитка для мастерской цветных бетонных кашпо ручной работы. Проект построен на React, TypeScript, Vite и Tailwind CSS; его можно без дополнительной настройки опубликовать на Netlify.

## Быстрый старт

```bash
npm install
npm run dev
```

Vite выведет локальный адрес сайта. Для production-сборки и её локального просмотра используйте:

```bash
npm run build
npm run preview
```

Проверка стиля кода:

```bash
npm run lint
```

## Публикация на Netlify

Файл `netlify.toml` уже задаёт команду `npm run build`, папку публикации `dist` и SPA-redirect. Создайте новый сайт в Netlify, подключите репозиторий и оставьте автоматически найденные настройки. После подключения домена замените `https://betonina.example/` в `index.html`, `public/robots.txt` и `public/sitemap.xml` на настоящий URL.

## Контент и фотографии

* Все тексты, ссылка на Telegram и массивы изображений собраны в `src/data/content.ts`.
* Ссылка заказа меняется через константу `telegramUrl` в этом файле.
* Главное изображение и порядок слайдов задаёт массив `heroImages`; первым элементом будет стартовый кадр.
* Для замены фото положите файлы в `public/images/hero/`, `public/images/works/`, а также замените `public/images/sizes.jpg` и `public/images/artist.jpg`. Ожидаемые имена перечислены в `src/data/content.ts`.
* Чтобы добавить работу в галерею, положите новый файл по шаблону `public/images/works/work-10.jpg` и увеличьте количество элементов в `worksImages`. Порядок определяется номерами файлов в этом массиве.

В репозитории предусмотрены выразительные цветовые заглушки, поэтому интерфейс остаётся целостным, если финальные фотографии ещё не добавлены. Перед публикацией замените их на настоящие JPG, PNG или WebP фотографии кашпо.

## Что реализовано

Сайт включает адаптивное меню, crossfade-слайдер первого экрана, masonry-галерею с доступным lightbox (Escape, стрелки и блокировка прокрутки), анимации появления через IntersectionObserver и поддержку `prefers-reduced-motion`. Метаданные, JSON-LD, robots, sitemap и Netlify-конфигурация подготовлены заранее.
